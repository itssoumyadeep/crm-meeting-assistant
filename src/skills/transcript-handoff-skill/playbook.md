# Transcript Handoff Playbook

Use this playbook when converting a transcript into a downstream-ready handoff.

## What to Extract

- Who attended the meeting.
- What problem or opportunity was discussed.
- What commitments were made.
- What the next follow-up actions should be.

## Writing Style

- Short, crisp, and factual.
- Focus on business meaning, not transcript noise.
- Use direct language that can be consumed by CRM or sales operations teams.

## Quality Bar

- Every item should be traceable to the transcript.
- If something is unclear, label it as uncertain rather than guessing.
- If there are no clear action items, return an empty list instead of inventing them.
